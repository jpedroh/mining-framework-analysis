Contributing

The source code for Billy is available on GitHub.

Fork and Pull Request

If you want to add a new feature or have a fix for some existing bug, you should first create your own fork. After that you can create your feature or bugfix in your fork. When your confident that your work is done, you should submit a Pull Request and await for someone to review your work. Feel free to ping someone if you get no response.
